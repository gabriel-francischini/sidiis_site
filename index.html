Digitação

O processo de digitação inicia-se na digitalização de uma imagem. Após a imagem ter sido digitalizada e salva em nossos servidores, ela é processada em lotes no Teleform Reader. É importante notar que os lotes não têm relação alguma com a separação dos arquivos digitais.

Após o processamento dos lotes no Teleform Reader, cada lote é encaminhado separadamente como um job para ser feito no Teleform Verifier. Cada job no Teleform Verifier passam por três fases diferentes:

  • Job QC
  • Evaluating
  • Ready for Correction

Job QC

Durante a fase de Job QC, um operador verifica se o software detectou corretamente o tipo de documento (se é um formulário de pesquisa, uma ficha médica, uma nota fiscal, etc). Caso não seja o tipo de documento adequado, é possível forçar a reidentificação do documento para o tipo correto.

Evaluating

É o momento que um OCR é usado para fazer o reconhecimento das informações preenchidas, de acordo com um formulário previamente inserido no sistema que indica os campos em que as informações podem ser inseridas.
